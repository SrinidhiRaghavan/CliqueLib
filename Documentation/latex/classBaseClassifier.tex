\subsection{Detailed Description}
\hyperlink{classBaseClassifier}{Base\+Classifier} provides an interface that each Armadillo-\/based algorithm in the library implements. Each algorithm constains at least a train function and a predict function that needs to be implemented by that particular algorithm.

The train function is virtual and is implemented by every Armadillo-\/based algorithm in Clique\+Lib. The train function accepts a matrix of sample inputs, a column vector of sample labels, and a uword representing the number of epochs. Given these inputs, the algorithms are able to generate a model for predicting on new inputs.
