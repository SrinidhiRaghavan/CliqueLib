Detailed Description

The Dataset structure is a convenient implementation constituting the basic components of a Dataset. Each dataset contains a matrix of features that is used for training and a vector column that is used for training. Similarly, the dataset further has a testing component that represents a matrix of features and a vector column of labels that needs to be predicted by a selected algorithm and tested.
